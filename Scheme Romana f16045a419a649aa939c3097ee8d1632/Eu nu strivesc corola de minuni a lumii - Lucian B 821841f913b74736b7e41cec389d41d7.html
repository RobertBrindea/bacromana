Eu nu strivesc - text programatic, autoportret liric

  • Monolog confesiv
  • Rolul eului liric

1a Expresionism (eu)
  • Prin: valorificarea intuitiei si afectivitatii (ca instrumente de cunoastere)
  • Eu poetic exacerbat -> elan vitalist de dimensiuni cosmice
  • Contopire cu misterul universal
  • Idei principale:
    • Poezie = forma de cunoastere
    • Relatia eului cu misterul
    • Raportul poetului cu lumea (mister) =>
      • Eul vrea sa amplifice taina, nu sa expliciteze
      • “Eu” (apare in vers 1, ultim si ½) => forta demiurgica de a amplifica misterul prin creatie (“asa imbogatesc si eu intunecata zare”)
1b Structura (monobloc, cauzalitate), Metaforizare, tehnici moderne
  • Monobloc + Rel de cauzalitate (1ul vers + ultimul vers => a iubi = a cunoaste) => coeziune
  • Elem de prozodie (absenta strofelor, versuri inegale(2-13), vers liber, ingambament, fluidizare („și nu ucid /cu mintea tainele, ce le-ntâlnesc/ în calea mea”) + metaforizare => modernism

3a Titlu (eu-lumea), metaf (corola)
  • Fixeaza 2 poli (eul - lumea)
  • Metafora “corola de minuni a lumii”
    • raportul special creator-univers de inspiratie
    • Preferinta pt metafora, analizata in “Despre geneza metaforei si sensul culturii”
      • Metaf != fig de stil, metaf = categorie filozofica
      • Aparitia metaforei legata de universul misterios, imposibil de explicat => rolul revelator al metaforei
      • Astfel, metaf:
        • Plasticizanta
        • Revelatorie (adanceste semnificatii) (preferata)
    • Corola = circularitate, armonie, frumusete, univers perfect cu om in centru
    • De minuni -> carac misterios, tainic al frumuseti
    • => lume = frumusete dar si fragilitate
  • Accentul: pe eu: plasat in poz cheie in poezie => unicitatea creatorului, rol esential
3b Opozitie (lumina mea - a altora)(luciferic-paradisiac)
  • Anticipeaza ideile din “Trilogia Cunoasterii”
(lumina = cunoastere) Lumina mea“Eu”: “nu strivesc”, “nu ucid (cu mintea)”, “sporesc”, “imbogatesc”, “iubesc”= cunoastere intuitiva, sensibila, acceptarea misterului universal (cunoastere luciferica) => asumarea misterului, potentarea lui, confundarea eului cu acesta => arta
Lumina altora“Altii”: “sugruma”, “ucid”Cunoastere rationala, logica, distrug misterul fara sa il rezolve (cunoastere paradisiaca = stiinta)

2a Enum tainelor
  • Enumeratia tainelor -> explicitarea metaforei din titlu
  • Per ansamblu = diversitatea lumii, a misterului in forme concrete, materiale
  • Individual: Enumeratie in crescendo al adancirii semnificatiilor
    • Flori = spatiul natural
    • Ochi = universul spiritual
    • Buze = universul rostirii cuv sau a iubirii
    • Mormite = experienta mortii => moartea cel mai mare mister
2b Comparatie luminea mea - a lunii
  • Capacitatea de a potenta misterul
  • Lumina lumii “nu micsoreaza… taina noptii”
  • “Imbogatesc… cu largi fiori de sfant mister”
    • “Sfant” - caracter sacru al misterului lumii
  • Camp lexical -> acc fumusetea universului, anulata de c paradisiaca
  • Precum razele lumii care arata si ascund, cunoasterea luciferica este situata intre fanic si criptic
